### For Voters

#### How to Vote Successfully
```
1. Have your phone ready
2. Ensure good internet connection
3. Open camera (for QR) or be ready to type code
4. Vote during designated time
5. Double-check your selection
6. Submit once (no duplicates)
7. Enjoy watching results if enabled!
```

#### Troubleshooting for Voters
```
Problem: QR code won't scan
Solution: Use manual code entry instead

Problem: Code doesn't work
Solution: Check spelling, try again

Problem: Can't see results
Solution: Admin may have disabled results view

Problem: Can't change vote
Solution: Re-voting may not be enabled
```
