## Getting Started

### Installation

1. Install all dependencies:
```bash
npm run install-all
```

2. Start the development server:
```bash
npm run dev
```

The application will run on:
- Frontend: http://localhost:5173
- Backend: http://localhost:3001

## Usage

### Creating a Poll

1. Navigate to the home page
2. Click "Create New Poll"
3. Enter poll title, options, and settings
4. Share the generated QR code or poll code with participants

### Voting

1. Scan the QR code or enter the poll code
2. Select your choice(s)
3. Submit your vote
4. See real-time results (if enabled)

### Viewing Results

1. Use the admin dashboard to monitor all active polls
2. View live analytics and vote counts
3. Export results when the poll closes
